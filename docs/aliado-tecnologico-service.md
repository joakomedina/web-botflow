# Servicio: Soy tu aliado tecnológico

Este documento define cómo implementar el servicio **"Soy tu aliado tecnológico"** en la web.

Incluye dos fases:

1. Card dentro de la página principal
2. Página detallada del servicio (landing)

El objetivo es convertir visitantes en leads, especialmente para campañas de Ads.

---

# Ubicación dentro de la página principal

La estructura actual de la home es:

Hero  
Problema  
Solución  
Servicios  
Proyectos  
Proceso  

El bloque **Soy tu aliado tecnológico** debe colocarse **entre Solución y Servicios**.

Estructura final:

Hero  
Problema  
Solución  
⭐ Soy tu aliado tecnológico  
Servicios  
Proyectos  
Proceso

Este bloque funciona como **puerta de entrada para personas que no saben qué servicio necesitan**.

---

# FASE 1 — CARD EN LA HOME

## Objetivo

Captar la atención de autónomos o profesionales que sienten que necesitan ayuda tecnológica pero no saben exactamente qué servicio contratar.

Debe ser una card visualmente destacada respecto a las demás.

---

## Diseño visual

La card debe diferenciarse del resto de servicios.

Fondo sugerido:

gradient suave azul.

Ejemplo Tailwind:

bg-gradient-to-br from-indigo-50 to-blue-50

Borde:

border border-indigo-200

Esto hace que el servicio destaque visualmente.

---

## Contenido de la card

### Microtexto

---

# PROBLEMA

Muchos profesionales utilizan herramientas digitales, pero no tienen un sistema que conecte todo.

Esto provoca:

- pérdida de tiempo en tareas manuales
- contactos desorganizados
- oportunidades que se pierden
- demasiadas herramientas sin coordinación

---

# SOLUCIÓN

Este servicio construye la base digital del negocio.

Incluye cuatro elementos principales.

---

# 1. Web profesional inicial

Landing profesional que permita explicar tu servicio y captar clientes.

Incluye:

- página web tipo landing
- formulario de contacto
- SEO básico
- despliegue

Objetivo:

tener presencia digital clara y profesional.

---

# 2. Sistema de captación de contactos

Cuando alguien contacta desde la web:

- llega al correo
- se registra el contacto
- se organiza el lead

Objetivo:

no perder oportunidades.

---

# 3. Automatización inicial simple

Se implementa una automatización sencilla para eliminar tareas repetitivas.

Ejemplos:

- formulario → email automático
- contacto → registro automático
- respuesta automática inicial

Nota:

solo se incluye **una automatización simple inicial**.

---

# 4. Sesión estratégica de diagnóstico

Sesión donde analizamos:

- procesos del negocio
- oportunidades de automatización
- posibles sistemas a implementar

Resultado:

un mapa de mejoras tecnológicas futuras.

---

# IDEAL PARA

Este servicio está pensado para:

- coaches
- consultores
- profesionales independientes
- autónomos
- pequeños negocios

---

# QUÉ NO INCLUYE

Para mantener el servicio claro, este paquete no incluye:

- automatizaciones complejas
- desarrollo de aplicaciones
- agentes de IA
- integraciones avanzadas

Estos elementos pueden desarrollarse en proyectos posteriores.
