# Publishing in the English blog

Each post is a `.md` file inside `content/blog/en/`.

Use this frontmatter:

```md
---
title: "Article title"
date: "2026-03-08"
excerpt: "Short summary for listing, SEO, and social previews."
tags:
  - "Automation"
coverImage: "/images/blog/your-image.jpg"
---
```

Then write the article body in Markdown.
