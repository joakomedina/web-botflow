# SERVICES / INTEGRATIONS

## Resumen

La V1 no depende de APIs externas obligatorias para renderizar contenido. Las integraciones activas son de contacto directo.

## Integraciones actuales

- WhatsApp
  - Uso: iniciar conversación desde CTA y formulario.
  - Implementación: enlace `https://wa.me/<numero>?text=<mensaje>`.
- Email
  - Uso: canal alternativo desde formulario y footer.
  - Implementación: enlace `mailto:`.

## Servicios/tecnologías referenciadas en proyectos

Estas tecnologías aparecen en portafolio como experiencia implementada:

- n8n
- Twilio
- Docker
- Coolify
- APIs de terceros
- Python / pandas / SQL
- Supabase
- FlutterFlow

## Variables configurables relacionadas

- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_CONTACT_EMAIL`
- `NEXT_PUBLIC_WHATSAPP_NUMBER`

## Posibles extensiones futuras

- Endpoint API para envío de formularios persistente.
- Integración con CRM para leads entrantes.
- Integración con sistema de agenda/reuniones.
