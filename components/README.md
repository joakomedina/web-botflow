# components/

Componentes UI reutilizables del proyecto.

## Convenciones

- Componentes de presentación sin lógica de datos acoplada.
- Props tipadas con TypeScript.
- Se usan server components por defecto.
- Solo se usa `"use client"` cuando hay interacción o animación.

## Organización

- Estructurales: `Navbar`, `Footer`, `PageHero`, `SectionTitle`
- Home: `Hero`, `ProblemSection`, `SolutionSection`, `ServicesGrid`, `ProjectsPreview`, `ProcessSection`, `AboutSection`, `CTASection`
- Cards: `ServiceCard`, `ProjectCard`
- Interacción: `ContactForm`, `Reveal`
